FileChooserPlugin

open class FileChooserPlugin(activity: ComponentActivity, cacheDir: File? = null, authority: String? = null)

Allows the user to choose a file on the device.

If cacheDir or authority are omitted, camera becomes unavailable since FileChooserPlugin cannot save temporary files with captured images.

Parameters

activity

The activity that starts intents.

cacheDir

The directory to store files captured by camera activity.

authority

The authority of the content provider that provides access to cacheDir for camera app.
